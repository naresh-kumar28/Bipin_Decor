# Project: Bipin Interiors (Full Stack Website)

Build a complete **production-ready full stack web application** for a client named **"Bipin Interiors"**.

This is a **service-based interior installation website**, NOT a product selling website.

This is strictly a **service-based interior installation and portfolio website**. It is NOT an e-commerce or product-selling site. 
Focus entirely on showcasing finished results, transformations (Before/After), and building client trust.

---

## 🎯 Main Objective

The website should showcase:
- Interior installation services
- Real project work (portfolio)
- Before/after transformations
- Customer reviews and ratings

Focus on **RESULTS (finished work)**, not products.

This website should look **premium, modern, clean, and highly attractive**, better than typical interior websites.

---

## 🧱 Tech Stack

### Frontend:
- React
- Tailwind CSS

### Backend:
- Django
- Django REST Framework

### Rules:
- Use **function-based views only**
- Use `request.data.get()` for handling request data
- Do NOT implement authentication (no login/signup)
- Keep backend simple and API-focused

---

## 🎨 Design Requirements

- Mobile-first responsive design
- Clean, minimal, modern UI
- Premium look (luxury interior feel)
- Proper spacing, typography, alignment
- Smooth hover effects and transitions

### Theme:
- Support both **Dark Mode and Light Mode**
- Add toggle button in header
