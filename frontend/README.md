# Bipin Interiors - Premium Decor Studio

A luxury interior design and installation website built with React and Vite.

## Features
- **Modern UI/UX**: Premium aesthetic with luxury brand colors (Gold & Slate).
- **Dark Mode**: Fully functional dark/light mode toggle.
- **Responsive Design**: Optimized for all devices.
- **Dynamic Assets**: Centralized asset management for images and videos.

## Tech Stack
- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React & Iconify

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/bipin-interiors.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file from the example:
   ```bash
   cp .env.example .env
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Generate a production-ready build:
```bash
npm run build
```

## Project Structure
- `src/assets`: Images, videos, and centralized asset exports.
- `src/components`: Reusable UI components (Navbar, Footer, etc.).
- `src/pages`: Application pages (Home, About, Services, Portfolio, etc.).
- `src/index.css`: Global styles and Tailwind configuration.
